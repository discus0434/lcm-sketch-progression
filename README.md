# LCM Sketch Progression

**LCM Sketch Progression is a web app that allows you to generate a sketch progression from your sketch.**

<p align="center">
  <img src="./assets/sample.gif" width=85%>
</p>

## Installation

### 1. Build the Docker images

```bash
make build
```

### 2. Run the app

```bash
make up
```

### 3. Access the app

Open `http://<PUBLIC_IP or 0.0.0.0>:10356` in your browser.

## Requirements

- Computer with more than 6GB of VRAM
- Docker
